# Tailwind Design System Visualizer

A modern interactive playground for exploring and understanding the Tailwind CSS design system in a visual, intuitive way.

This project provides developers with a powerful environment to inspect spacing, typography, layout behavior, utilities, and design tokens directly in the browser. Instead of memorizing classes or scanning documentation, users can visually explore how Tailwind works.

The goal of this tool is to make learning and designing with Tailwind faster, more practical, and more enjoyable.

## Features

- Visual exploration of Tailwind design tokens
- Interactive spacing scale inspector
- Color palette browser
- Typography preview system
- Layout and container visualization
- Positioning and z-index playground
- Effects and utility demonstrations
- Tailwind utility cheatsheet
- Component inspection overlay
- Built-in search for utilities

## Why This Project Exists

Tailwind is powerful, but understanding the full design system behind its utilities can take time.

This visualizer turns the design system into an interactive interface where developers can experiment, learn, and prototype much faster.

Instead of reading documentation, you can see the system in action.

## Getting Started

Clone the repository:

```
git clone https://github.com/USERNAME/REPO_NAME.git
```

Install dependencies:

```
npm install
```

Start development server:

```
npm run dev
```

Build production version:

```
npm run build
```

Preview the build:

```
npm run preview
```
